### 🏆 Featured AI & Automation Showcases

```bash
$ git log --graph --pretty=format:'%h - %an : %s' --stat
```

* **🤖 [TV Trade](https://github.com/ik123a/tv-trade)** — AI-Powered Automated TradingView Trading Extension
  * *Features:* Integrated **16 concurrent LLM models** (9 NVIDIA NIMs + 7 OpenRouter) for trade signal analysis. Completely localized typography layout, fully portable style system, auto-calculates SL/TP, and records detailed trading logs on an offline dashboard.
  * *Stack:* JavaScript, Chrome Extension APIs, CSS3, HSL design system.

* **⚡ [GFG NVIDIA Auto](https://github.com/ik123a/gfg-nvidia-auto)** — GeeksforGeeks Multi-Model Autopilot Solver
  * *Features:* Automated solver that scrapes problem definitions, compiles/submits code, and executes a **Multi-Model Consensus & LLM Judge** pipeline. Features **Self-Correction & Evolution** logic with up to 25 context-aware repair cycles on compiler/runtime errors.
  * *Stack:* JavaScript (Extension), Python, Flask (Backend Server), NVIDIA NIM API (Llama 3.3, Qwen 2.5 Coder, Nemotron).

* **🎯 [Leetcode NVIDIA Auto](https://github.com/ik123a/leetcode-nvidia-auto)** — Headless Leetcode Autopilot Solver
  * *Features:* Full browser automation leveraging **Playwright** paired with a Flask backend. Automatically parses challenges, writes solutions, compiles, handles test cases, and registers submissions with multi-model consensus.
  * *Stack:* Python, Playwright, Flask, NVIDIA NIM API.

* **🩻 [RSNA Pneumonia Detection](https://github.com/ik123a/rsna-pneumonia-detection-challenge)** — Faster R-CNN Object Detection in Chest X-Rays
  * *Features:* Advanced medical computer vision pipeline built using PyTorch. Custom Region Proposal Network (RPN) anchor tuning specifically optimized for lung lesions, utilizing AdamW with StepLR and Early Stopping. Optimized with Automatic Mixed Precision (AMP) to reduce memory footprints.
  * *Stack:* PyTorch, Torchvision, Python, Jupyter Notebook.

* **🛡️ [Deepfake Image Detection](https://github.com/ik123a/deepfake-detection)** — Premium Image Classification Pipeline
  * *Features:* Image classification pipeline utilizing pre-trained **EfficientNetB0**. Corrected Keras's built-in image double-rescaling resolution issue, featuring smart Laptop/Colab dual runtime execution modes.
  * *Stack:* TensorFlow, Keras, OpenCV, Scikit-learn, Python.

* **🔬 [ResNet-50 CIFAR Comparison](https://github.com/ik123a/resnet50-transfer-learning-cifar)** — PyTorch Transfer Learning Benchmark
  * *Features:* Comprehensive study evaluating **three distinct network freezing strategies** (Fully Frozen, Partially Frozen, and Fully Trainable). Features stable Batch Normalization tracking in frozen layers to preserve ImageNet baseline stats.
  * *Stack:* PyTorch, Python, Jupyter Notebook.
